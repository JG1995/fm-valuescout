# Squad

One of the main features that makes this app unique is that it will actively incorporate your squad into comparisions. The user is encouraged to, when first using the tool, to select which club they are currently managing. If they have loaded a database, either through CSV or through memory, the squad will then be populated.

The main feature is the tactics board. The user chooses what formation their club is playing as - Either presets, or they can manually move the "pins" around the pitch" into areas representing the available positions in Football Manager. For each position, there will then be some preset "Archetypes" that the user can choose from - For example, a DM may have "defensive midfielder" or "deep lying playmaker", whereas a AMR might have "creative winger", "goalscoring winger" or "goalscoring inside forward", or a striker would have "creative forward", "target forward" or "poacher". These Archetypes would come with the app, and score players based on pre-selected and pre-weighted metrics.

Each position should have both an in-possession and out-of-possession archetype.

The user is however encouraged to create custom Archetypes, which will make a modal appear where they themselves choose the exact metrics and weighting of them, to calculate the player's suitability to a given role. The user can either create one from scratch, or create one based on an existing Archetype, either the default ones or another custom Archetype.

With the tactics board, the app will also suggest an optimal lineup for the user, based on the metrics of each player in the squad. The user can choose if they want suggestions based purely on metrics, or to also factor in the actual in-game ability of the player.

Alongside the tactics board, is also a squad overview, showing you the squad in a more table-like view, similar to what you find in the Football Manager game. Colour-coded metrics are also shown here, with the exact ones being shown depending on which roles the user has selected on the tactics board. The user can then tab between Senior Squad, U21, U18 or all players.
